# 🍎 YS Video Downloader - macOS Kurulumu

## 📥 İndirme

[Releases](https://github.com/ysonmezer/ysvdown/releases/latest) sayfasından `ysvdown_v2.6_macos.dmg` dosyasını indirin.

## 🚀 Kurulum ve Kullanım

### 1. DMG'yi Açın
İndirilen DMG dosyasına çift tıklayın.

### 2. Terminal'i Açın
- Spotlight (⌘ + Space) → "Terminal" yazın
- Enter'a basın

### 3. Şu Komutları Çalıştırın
```bash
cd "/Volumes/YS Video Downloader v2.6"
./YS\ Video\ Downloader\ v2.6
```

Program açılacak!

## ⚠️ İlk Çalıştırmada

macOS güvenlik uyarısı verebilir:

**Çözüm:**
1. System Preferences (Sistem Tercihleri) → Security & Privacy (Güvenlik ve Gizlilik)
2. "Open Anyway" (Yine de Aç) butonuna tıklayın

## 📖 Kullanım

1. Video linkini yapıştırın
2. Format seçin (Video/MP3)
3. Kalite seçin
4. İndir butonuna tıklayın

## 🐛 Sorun Giderme

### "ffmpeg bulunamadı" hatası
DMG içindeki tüm dosyalar orada olmalı. DMG'yi yeniden indirin.

### Terminal komutu çalışmıyor
Komutları **tam olarak** kopyalayın (boşluklar ve \ işaretleri önemli).
